# 假目标/诱饵综述项目设计

## 一、项目定位

本项目以 dummy/decoy 为核心对象，建设一个覆盖古往今来、跨物理实体/信号/网络空间的假目标与诱饵大全。现有“自主式假飞机系统”文档作为重点应用方向之一，而不是唯一主线。研究对象包括历史诈术、静态假目标、充气诱饵、主动式假目标、电磁诱饵、多谱段欺骗系统、网络诱捕、可消耗自主诱饵平台，以及面向机场、高价值装备、分布式作战节点和网络空间的综合欺骗体系。

核心目标不是单纯汇总案例，而是回答三个问题：

1. 从古代诈术到智能无人系统，假目标/诱饵的底层逻辑有哪些连续性？
2. 假目标如何从实体“形似”扩展到信号、行为、网络身份和体系叙事？
3. 现代 ISR、精确打击、AI 识别和网络攻防条件下，假目标/诱饵为什么重新变得关键？
4. 自主式假飞机系统应如何在更大的 dummy/decoy 谱系中定位？

## 二、建议题目

推荐主标题：

《假目标与诱饵系统大全：从古代诈术到多谱段自主欺骗》

备选标题：

- 《Dummy/Decoy 全谱系研究：实体、信号、网络与智能化诱饵》
- 《透明战场下的欺骗能力重构：假目标/诱饵技术发展与应用综述》
- 《从充气模型到自主式假目标：现代军事诱饵系统的作战价值与技术路线》
- 《面向机场生存力的主动假目标系统综述与自主式假飞机发展建议》

## 三、研究框架

### 1. 背景与基本逻辑

- 假目标/诱饵的核心不是“做假物体”，而是操纵观察者、传感器、算法、武器或入侵者的判断与资源分配。
- 古代诈术、二战假军队、冷战导弹/雷达诱饵、俄乌战场充气装备、网络蜜罐和自主无人诱饵，本质上都服务于“制造错误信念”。
- 不同时代的差异主要体现在被欺骗对象、传感器体系、目标价值、成本交换和评估方式。

### 2. 现代战场背景

- 低轨卫星、无人机、SAR、红外、电子侦察和 AI 识别提高目标暴露概率。
- 远程精确打击压缩机场、指挥所、炮兵阵地、装甲集群等高价值目标的生存窗口。
- 俄乌冲突显示，假目标能有效干扰侦察打击链并改变成本交换关系。

### 3. 概念谱系

- 伪装：降低可探测性。
- 隐蔽：减少暴露机会。
- 假目标：制造可被攻击的虚假对象。
- 诱饵：主动吸引侦察、锁定或打击。
- 信号诱饵：制造虚假的电磁、热、声、雷达、通信或协议特征。
- 网络诱饵：通过蜜罐、蜜标、假凭证、假服务和假数据吸引、延迟或观察入侵者。
- 主动欺骗系统：通过外形、热、雷达、电磁、行为、网络信号和情报叙事构造完整欺骗图像。

### 4. 分类维度

- 时间维度：古代/前工业、工业化、两次世界大战、冷战与精确打击时代、当代透明战场、智能化未来。
- 特征维度：实体、信号、行为、网络、叙事/情报。
- 物理维度：经典式、机械化、信息化、智能化。
- 技术维度：设计/生成、生产/制造、部署/运用、测试/评估。
- 领域维度：陆、空、海、潜、天、电磁频谱、网络、信息认知、跨域。
- 效果维度：吸引火力、消耗弹药、拖延决策、过载 ISR、掩护机动、干扰 BDA、暴露敌方传感器、收集网络攻击技战术。

### 5. 技术演进

- 第一代：视觉形似，木架、帆布、充气模型。
- 第二代：快速部署与低成本批量化，典型为充气坦克、火炮、飞机模型。
- 第三代：多谱段特征模拟，加入热源、雷达反射器、电磁发射等。
- 第四代：主动式/无人化，具备遥控或自主移动、动态行为模拟、集群协同。
- 第五代：认知与体系欺骗，面向敌方 ISR-打击闭环或网络攻击链注入连续、可信、可验证的虚假图像。

### 6. 典型案例

- 古代/前工业诈术：假营地、疑兵、假撤退、火光和旗帜欺骗。
- 二战坚忍行动：假军队、假登陆艇、无线电欺骗和双重间谍共同构造虚假战役叙事。
- Sea Wolf F-35 充气诱饵：两人约 10 分钟部署，具备热特征和雷达反射能力。
- 俄乌战场假坦克/假火炮/假 HIMARS：用低成本目标诱导昂贵弹药消耗。
- Temerland 主动假坦克：基于普通车辆，远程控制，强调机动欺骗。
- 美军 Replicator：体现低成本、可消耗、自主系统快速规模化的趋势。
- 电磁诱饵和假 C2 节点：通过模拟通信、电磁和指挥节点信号误导敌方侦察。
- 网络蜜罐/蜜标：通过可交互假系统、假凭证和假数据吸引入侵者，提升告警质量并收集攻击行为。

### 7. 技术路线矩阵

| 维度 | 低端方案 | 中端方案 | 高端方案 |
|---|---|---|---|
| 外形 | 木架/帆布 | 充气/折叠结构 | 复合材料骨架+智能蒙皮 |
| 光学 | 涂装 | 伪装网/局部细节 | 可变光学表面 |
| 红外 | 简单热源 | 分区薄膜加热 | 动态热图/主动冷却 |
| 雷达 | 金属反射器 | 角反射器/伦伯透镜 | DRFM/动态 RCS 整形 |
| 电磁 | 无 | 简单信号发射 | 模拟通信/数据链/雷达工况 |
| 行为 | 静态 | 遥控移动 | 自主机动/集群协同 |
| 网络 | 简单蜜罐 | 蜜标/假凭证/假服务 | 可编排欺骗网络/高交互诱捕环境 |
| 评估 | 人工判断 | 传感器回传 | 闭环欺骗效果评估 |

### 8. 自主式假飞机系统专题

重点围绕机场和分布式航空作战展开：

- 模拟停机、维护、滑行、转场、起飞准备等行为链。
- 在真实战机周边构造真假混编目标群。
- 通过热、雷达、电磁和灯光信号制造可被敌方多源情报交叉验证的假象。
- 以低成本目标吸收敌方昂贵弹药，降低真实战机暴露与损失概率。

### 9. 关键问题与风险

- 多谱段一致性：外形、热、雷达、电磁和行为不能互相矛盾。
- 跨域一致性：实体、信号、网络和情报线索不能互相矛盾。
- 成本约束：高保真不能导致成本接近真实装备。
- 能源与热管理：主动加热、信号发射和自主机动会显著增加功耗。
- 部署速度：系统必须匹配机场高节奏作战。
- 抗风与结构可靠性：大尺寸轻量结构在野外环境下风险高。
- 网络诱饵暴露：低交互蜜罐、默认配置、时序特征和服务指纹容易被识别。
- 对抗升级：敌方 AI 识别会持续学习假目标特征。

## 四、建议最终交付件

1. 假目标/诱饵大全 README 与结构化知识库。
2. 汇报 PPTX，约 20-30 页。
3. 案例数据库 XLSX/CSV，字段包括时代、领域、对象、特征、部署方式、效果机制、证据强度、来源。
4. 分类体系与技术路线图，覆盖古代至 2035 年发展趋势。
5. 自主式假飞机系统专题章节，可直接服务后续立项论证。
6. 网络诱饵专题章节，覆盖 honeypot、honeytoken、deception network 和 cyber range。
