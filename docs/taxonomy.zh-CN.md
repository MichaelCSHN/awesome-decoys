# 假目标/诱饵分类体系

[English](taxonomy.md) | 中文

## 工作定义

假目标/诱饵是人为构造的对象、信号、行为、身份、系统或叙事，目的在于让观察者、传感器、算法、武器、入侵者或决策者形成错误判断，或错误分配资源。

相关概念：

- **伪装**：降低真实目标可见性。
- **隐蔽**：减少被发现机会。
- **假目标**：制造虚假的可观察对象。
- **诱饵**：主动吸引观察、锁定、交互或打击。
- **欺骗行动**：把实体、信号、行为和叙事组织成完整的虚假图像。

## 时间维度

- **古代/前工业**：疑兵、假营地、火光、旗帜、佯退、诈术。
- **工业化时代**：假炮、假船、假机场、假工厂、伪装学校。
- **两次世界大战**：充气坦克、假登陆艇、无线电欺骗、假军队、双重间谍。
- **冷战与精确打击时代**：雷达诱饵、导弹突防辅助、机动发射车、SAM 诱骗、电子战。
- **当代透明战场**：无人机观察下的充气假目标、假 C2、假雷达、多谱段特征、成本交换。
- **智能化未来**：自主诱饵平台、自适应信号、诱饵蜂群、网络-物理欺骗、闭环效果评估。

## 特征维度

- **实体/物理**：坦克、飞机、舰艇、火炮、导弹、雷达、C2、桥梁、仓库、人员形体等。
- **信号**：无线电、雷达反射、RF 信标、热源、声学、灯光、AIS/GPS 类信号、合成回波。
- **行为**：移动、维修、补给、发射准备、巡逻节奏、机场运作节奏、pattern-of-life。
- **网络**：蜜罐、蜜网、蜜标、假凭证、假服务、假数据库、假身份和诱捕环境。
- **叙事/情报**：假战斗序列、控制泄露、双重间谍、假后勤轨迹、假文件。

## 物理演进维度

| 阶段 | 核心特征 | 典型能力 |
|---|---|---|
| 经典式 | 人工构造、视觉为主、本地使用 | 木架、帆布、火光、假营地、假旗帜 |
| 机械化 | 快速部署、可搬运、可重复使用 | 充气结构、折叠框架、加热器、反射器、简单发射器 |
| 信息化 | 可编程、联网、协同 | RF 发生器、远程控制、网络化发射器、传感器触发 |
| 智能化 | 可移动、可感知、可适应 | 无人平台、自主路径、动态热/RF 控制、蜂群协同 |

## 技术生命周期

- **设计/生成**：目标特征建模、威胁传感器建模、CAD/数字孪生、合成图像、RCS/热/声/网络特征设计。
- **生产/制造**：充气织物、轻量框架、涂层、加热器、反射器、RF 载荷、低成本电子模块。
- **部署/运用**：布设、间隔、时机、激活、移动、供电、抗风、与真目标协同。
- **测试/评估**：EO/IR/SAR/RF/声学/网络测试、红队分类、成本交换、BDA 模糊度、网络交互深度。

## 其他维度

- **领域**：陆、空、海、潜、天、电磁、网络、信息认知、跨域。
- **欺骗对象**：人眼、无人机操作员、卫星分析员、雷达处理器、导弹导引头、AI 分类器、网络入侵者、指挥员。
- **效果机制**：吸引火力、消耗弹药、拖延决策、掩护机动、干扰 BDA、暴露敌方传感器、收集攻击者技战术。
- **保真度**：视觉级、单特征、多谱段、行为级、联网级、叙事级。
- **成熟度**：概念、历史案例、原型、训练器材、商业产品、部署系统、战场观察、已验证效果。
- **反诱饵角度**：光学异常、热不一致、SAR 散射、RF 指纹、pattern-of-life、多源融合、网络指纹。
