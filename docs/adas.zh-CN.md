# 自主式假飞机系统（ADAS）专题

[English](adas.md) | 中文

ADAS 是本项目的重点应用方向之一，面向机场生存力和分布式航空作战，通过低成本假飞机和配套信号/行为构造可信的假空中力量存在。

## 作战问题

机场、停机坪、临时起降点和分布式航空节点在卫星、无人机、SAR、红外和电子侦察下暴露风险高。真实飞机价值高、数量有限、维修和补给链复杂，因此假飞机系统可以通过制造目标不确定性来降低真实资产风险。

## 主要作用

- 构造真假混编停机图像。
- 消耗敌方 ISR 时间和精确弹药。
- 诱导敌方错误 BDA。
- 支撑敏捷战斗运用和分布式机场体系。
- 在真实机动和维护活动周边制造合理噪声。

## 多谱段可信栈

| 层级 | 最低有效特征 | 高端特征 |
|---|---|---|
| 光学 | 外形、尺度、涂装、阴影 | 可配置型号、维护姿态、细节表面 |
| 热 | 发动机区域热源、环境热差 | 分区热图、时间序列热行为 |
| 雷达 | 反射器或 RCS 增强 | 可调反射/动态 RCS |
| 电磁 | 简单信标或 RF 特征 | 地勤、数据链、雷达/维护发射模拟 |
| 行为 | 静态停放、周期换位 | 遥控/自主滑行样行为、灯光纪律、服务例程 |
| 系统 | 人工布设 | 队列调度、欺骗计划、传感器反馈和 BDA 收集 |

## 自主性边界

并非所有 ADAS 都需要完全自主。更现实的路线是逐步增加：

1. 快速部署与人工布设。
2. 遥控移动和签名控制。
3. 半自主调度和行为脚本。
4. 传感器触发的动态伪装。
5. 多平台协同和闭环效果评估。

## 主要风险

- 多谱段不一致。
- 能耗和热管理困难。
- 轻量大尺寸结构抗风能力不足。
- 成本接近真实装备后失去意义。
- pattern-of-life 不真实。
- 敌方 AI/ATR 持续学习假目标特征。
